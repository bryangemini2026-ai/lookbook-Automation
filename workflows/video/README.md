# Video Workflow Templates

Place ComfyUI Video Server workflow JSON files here.

## Recommended Workflows

| File | Purpose |
|------|---------|
| `animate_basic.json` | Basic AnimateDiff animation |
| `animate_zoom.json` | Zoom-in animation effect |

## Setup Requirements

1. ComfyUI Video Server (port 8288)
2. AnimateDiff custom node installed
3. SD1.5 checkpoint
4. AnimateDiff motion module (v3)
